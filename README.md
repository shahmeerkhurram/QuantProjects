# Quantitative Risk & Financial Modeling Projects

This repository showcases a set of quantitative finance projects focused on **risk modeling, derivatives pricing, and systemic risk analysis**.  
All projects are implemented using Python with an emphasis on **practical financial applications and mathematical rigor**.

---

## 🔹 Projects

### 1. Value-at-Risk (VaR) Modeling
**Description:**  
Developed a Value-at-Risk framework to estimate potential portfolio losses under different assumptions.

**Key Features:**
- Historical VaR
- Parametric (Variance-Covariance) VaR
- Comparison of methods under different market conditions
- Visualization of loss distributions and tail risk

**Skills:** Probability Theory, Statistical Modeling, Risk Analysis

---

### 2. Black-Scholes Option Pricing
**Description:**  
Implemented the Black-Scholes model for pricing European options and analyzing sensitivities.

**Key Features:**
- Call and Put option pricing
- Sensitivity analysis with respect to underlying price
- Visualization of option price behavior
- Numerical validation via simulation

**Skills:** Derivatives Pricing, Stochastic Modeling, Numerical Methods

---

### 3. Art Gallery Risk Optimization
**Description:**  
Applied the Art Gallery Problem from computational geometry to portfolio risk monitoring.

**Key Features:**
- Modeled assets as points in a geometric space
- Optimized placement of minimal “risk checkpoints”
- Simulated random shocks and analyzed coverage efficiency
- Visualized portfolio coverage using clustering techniques

**Skills:** Optimization, Computational Geometry, Risk Monitoring

---

### 4. Network Risk Propagation
**Description:**  
Modeled a financial portfolio as a network to analyze systemic risk and contagion effects.

**Key Features:**
- Graph-based representation of asset correlations
- Simulation of loss propagation across the network
- Multi-step contagion modeling
- Identification of high-risk (critical) nodes

**Skills:** Network Theory, Systemic Risk, Graph Modeling
