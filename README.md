# Quantitative Finance Projects

This repository showcases two quantitative finance projects demonstrating skills in **risk analysis**, **option pricing**, **numerical methods**, and **Python programming**.

## Projects

### 1. Value-at-Risk (VaR) Engine
- Computes Historical, Parametric, and Monte Carlo VaR for a portfolio
- Backtests results to check model accuracy
- Visualizes portfolio return distributions and VaR thresholds

### 2. Black-Scholes Option Pricing
- Prices European call and put options using the Black-Scholes model
- Computes Greeks: Delta, Gamma, Vega
- Visualizes price sensitivity to underlying asset and volatility

## Skills Demonstrated
- Python (NumPy, Pandas, Matplotlib, SciPy)
- Quantitative risk analysis
- Derivatives pricing and sensitivity analysis
- Numerical methods and data visualization
